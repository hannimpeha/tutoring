## 자료구조
- 내가 디자인한 / 복잡한 / 특정 기능을 가지고 있는 데이터 타입
- class 로 구현
    - class의 구성 요소
        - Attribute: 데이터들
        - Function / Method: 기능들
- 대표적인 자료구조
    - Linked List
        - element 들이 연결되어있는 리스트
    - Stack
        - "first in last out" 의 철학을 가진 리스트
        - push
        - pop
    - Queue
        - "first in first out" 의 철학을 가진 리스트
        - enqueue
        - dequeue
    - Graph
        - network
        - signed / weighted / directed graph
    - Tree
        - 상하관계가 뚜렷한 graph
        - parent / child

## String formatting
- 스트링을 만드는 규칙이라고 생각하면 됨.
- '패턴 변수1, 변수2, ...' % (값1, 값2, ...)
- '%d + %d = %d' % (1, 2, 3)
- '%d + %d = %d' % (1, 3, 4)
- %d: 정수
- %lf: float
- %s: string
- %r: bool
- %3d
- %.2lf
